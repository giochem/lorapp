# LOR MERN WEB

This is the LOR image store app

## Usage

"proxy": "http://localhost:8080",

Rename the .envexample to .env and your variable

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

### RUN Client

```
npm run client
```

### Run Fullstack

```
npm run dev
```
