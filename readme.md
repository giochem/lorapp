# LOR MERN app

This is the LOP image app

## Usage

<!-- Rename the .envexample to .env and add your MONGO_URI -->

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

### RUN Client

```
npm run client
```

### Run Fullstack

```
npm run dev
```
